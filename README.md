### Project Title
Bikeshare study in 3 different cities.

### Description
This is a python project in which the program asks the user for a city, month and day input, in order to check and get some statistics of the use of shared bikes.

### Files used
To run succesfully you will need the city files not included in this repository (chicago.csv, new_york_city.csv and washington.csv).
